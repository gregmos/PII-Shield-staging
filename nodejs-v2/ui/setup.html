Couldn't find the ZIP? Other ways to install
  1. Marker fallback — drop a small text file named .pii_shield_model_here in the same folder as the downloaded ZIP, then click Install downloaded ZIP again. PII Shield will scan your filesystem for the marker.
  2. Configure folder — Claude Desktop → Settings → Extensions → PII Shield → Model Downloads Folder. Point at the folder where your browser saves files, then click Install.
  3. Terminal one-liner (advanced) — see the README's Install model section for the PowerShell / bash one-liner that downloads and installs in one step.
